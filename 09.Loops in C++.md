## 2. **üî¢ For Loops**  
**Timestamp:** [00:02:00](https://youtu.be/_1AwR-un4Hk?t=120)  
### Syntax:  
```cpp  
for (initialization; condition; iteration) {  
    // Code to repeat  
}  
```  

### Example: Print "Hello World" 5 times  
```cpp  
for (int i = 0; i < 5; i++) {  
    std::cout << "Hello World üåç" << std::endl;  
}  
```  

### Breakdown:  
1. **üöÄ Initialization**: `int i = 0` (executes once at start).  
2. **‚úÖ Condition**: `i < 5` (checked before each iteration; loop continues if `true`).  
3. **üîÑ Iteration**: `i++` (executes after each iteration).  

### Key Points:  
- **üéõÔ∏è Flexibility**: All three parts (`initialization`, `condition`, `iteration`) can be modified.  
  ```cpp  
  int i = 0;  
  for ( ; ; ) {  
      if (i >= 5) break;  
      std::cout << "Hello World üåç" << std::endl;  
      i++;  
  }  
  ```  
- **‚ôæÔ∏è Endless Loop**: Remove condition ‚Üí `for (;;) { ... }`.

## 3. **üîÑ While Loops**  
**Timestamp:** [00:07:24](https://youtu.be/_1AwR-un4Hk?t=444)  
### Syntax:  
```cpp  
while (condition) {  
    // Code to repeat  
}  
```  

### Example: Equivalent to the For Loop  
```cpp  
int i = 0;  
while (i < 5) {  
    std::cout << "Hello World üåç" << std::endl;  
    i++;  
}  
```  

### Use Case:  
- **üéÆ Game Loop**:  
  ```cpp  
  bool isRunning = true;  
  while (isRunning) {  
      UpdateGame();  
      RenderGraphics();  
  }  
  ```

## 4. **‚û∞ Do-While Loops**  
**Timestamp:** [00:10:12](https://youtu.be/_1AwR-un4Hk?t=612)  
### Syntax:  
```cpp  
do {  
    // Code to execute at least once  
} while (condition);  
```  

### Example:  
```cpp  
int i = 0;  
do {  
    std::cout << "Hello World üåç" << std::endl;  
    i++;  
} while (i < 5);  
```  

### Key Difference:  
- Executes the body **at least once**, even if the condition is initially `false`.  

---

## 5. **üìä Key Differences**  
| Loop Type      | When to Use                          | Notes                               |  
|----------------|--------------------------------------|-------------------------------------|  
| **üî¢ For**     | Known number of iterations           | Compact syntax for index-based loops |  
| **üîÑ While**   | Condition-based repetition           | Flexible for dynamic conditions     |  
| **‚û∞ Do-While**| Execute code once before checking    | Rarely used; specific edge cases    |  

---

## 6. **üìù Summary**  
- **For Loops**: Best for fixed iterations (e.g., arrays, counting).  
- **While Loops**: Ideal for unknown iterations (e.g., game loops).  
- **‚ö†Ô∏è Avoid Infinite Loops**: Ensure the loop condition eventually becomes `false`.
